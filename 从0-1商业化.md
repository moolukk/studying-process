## 从0-1商业化

### 1.什么是商业化

· 用各种方式把业务、流量变成收入：降低获客成本，提升用户留存和活跃，提升变现效率

查询基础语句：

| 关键字          | 功能                | 示例语句                                                     |
| --------------- | ------------------- | ------------------------------------------------------------ |
| select          | 查询数据            | select * form  users                                         |
| delete from     | 删除某条数据        | delete from                                                  |
| from            | 指定数据来源表      | select name from users；                                     |
| where           | 条件筛选            | select * from users where age >18;                           |
| distinct        | 去重                | select distinct city from users                              |
| order by        | 排序                | select * form  users order by created_at desc;               |
| limit           | 限制返回条数        | SELECT * FROM users LIMIT 10;                                |
| offset          | 跳过前几条数据      | SELECT * FROM users LIMIT 10 OFFSET 20;                      |
| group by        | 按字段分组          | SELECT city, COUNT(*) FROM users GROUP BY city;              |
| having          | 分组后筛选          | SELECT city, COUNT(*) FROM users GROUP BY city HAVING COUNT(*) > 10; |
| join            | 多表连接            | SELECT u.name, o.amount FROM users u JOIN orders o ON u.id = o.user_id; |
| left join       | 左连接              | SELECT u.name, o.amount FROM users u LEFT JOIN orders o ON u.id = o.user_id; |
| on              | 指定连接条件        | …ON u.id = o.user_id                                         |
| as              | 字段或表的别名      | SELECT name AS 用户名 FROM users;                            |
| in              | 值在某个集合内      | SELECT * FROM users WHERE city IN ('Beijing', 'Shanghai');   |
| between         | 范围内查询          | SELECT * FROM users WHERE age BETWEEN 18 AND 30;             |
| like            | 模糊匹配            | SELECT * FROM users WHERE name LIKE 'Zhang%';                |
| is null         | 判断是否为空        | SELECT * FROM users WHERE phone IS NULL;                     |
| case when       | 条件判断语句        | SELECT name, CASE WHEN age > 30 THEN '老' ELSE '新' END FROM users; |
| count（）       | 计数                | SELECT COUNT(*) FROM users;                                  |
| sum（）         | 求和                | SELECT SUM(amount) FROM orders;                              |
| AVG（）         | 求平均值            | SELECT AVG(age) FROM users;                                  |
| MAX（）/MIN（） | 最大值/最小值       | SELECT MAX(age), MIN(age) FROM users;                        |
| UNION           | 合并两个查询结果    | SELECT name FROM A UNION SELECT name FROM B;                 |
| insert into     | 插入新数据          | INSERT INTO users (name, age) VALUES ('Tom', 25);            |
| update          | 更新数据            | UPDATE users SET age = 30 WHERE name = 'Tom';                |
| create table    | 创建一张表          | CREATE TABLE users (id INT, name VARCHAR(50));               |
| drop table      | 删除表              | DROP TABLE users;                                            |
| with            | 公用表表达式（CTE） | WITH temp AS (SELECT * FROM users WHERE age > 30) SELECT * FROM temp; |
| rank（）        | 窗口函数、排名      | SELECT name, RANK() OVER (ORDER BY age DESC) FROM users;     |
| over（）        | 窗口函数作用范围    | SUM(amount) OVER(PARTITION BY user_id)                       |
